# Snake Game 贪吃蛇游戏

## English Version

### Introduction

This is a simple implementation of the classic Snake game written in C#. The game features a snake that moves around a grid, consuming fruits to grow longer. The player controls the snake's direction using arrow keys or WASD keys, trying to eat as many fruits as possible without colliding with walls or the snake's own body.

### Features

- Classic Snake gameplay
- Four difficulty levels (Easy, Medium, Hard, Impossible)
- Score tracking
- Keyboard controls (Arrow keys or WASD)
- Collision detection (walls and self)
- Fruit generation that avoids snake body positions

### How to Play

1. Run the game executable
2. Choose a difficulty level from the menu
3. Control the snake using:
   - Arrow keys (↑, ↓, ←, →)
   - Or WASD keys (W, A, S, D)
4. Press Q to quit the game at any time
5. Try to eat as many fruits as possible without colliding

### Game Rules

- The snake grows longer when it eats a fruit
- The game ends if the snake hits a wall or itself
- The player wins if the snake fills the entire board

### Controls

- **Movement**: Arrow keys or WASD
- **Quit**: Q key
- **Play Again**: Y key (after game over)
- **Exit**: N key (after game over)
